# Archive

Decision records here are no longer current — `rejected`, `superseded`, `obsolete`, or
`deprecated`. They are kept in git for history but excluded from the published site.

Quarto does not render files or directories whose name begins with `_`, so nothing under
`_archive/` appears in the built site, and the Decisions listing (which pulls from
`decisions/**/*.md`) drops them automatically.

To republish a record, move it back into `decisions/`.
